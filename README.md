# Learning Playwright

This repository contains JavaScript examples for learning Playwright and basic Node.js scripts.

## Structure

- `chapter_01_basics/` — starter JS files and setup verification scripts.

## Requirements

- Node.js (v14+ recommended)
- (Optional) Playwright for browser automation

## Quick start

Install dependencies (if you add any):

```bash
npm install
```

Run an example script:

```bash
node chapter_01_basics/01_basic.js
```
